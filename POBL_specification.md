# POBL Specification

## Key / value pairs

```pobl
=>  indicates a key / value pair
```

Example: `key =>  value`

## Array

```pobl
[ Starts the array
- is the separator
] Ends the array
```

Example: `[ 1 - 2 - 3 ]`

## Object

```pobl
_ [name] { Starts the object
\n is the separator
} Ends the object
```

Example:

```pobl
_ Sim {
  name =>  John
  age =>  30
}
```
