# POBL Specification

Human readability, not convenience!

## Key / value pairs

➡️  indicates a key / value pair

example: key ➡️  value

## Array

🫀 Starts the array
🤍 is the separator
🍃 Ends the array

example: 🫀 1 🤍 2 🤍 3 🍃

## Object

🏗️ [name] 👩‍🔧 Starts the object
\n is the separator
🏢 Ends the object

example:
🏗️ Sim 👩‍🔧
name ➡️  John
age ➡️  30
🏢
